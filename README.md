# Interactive Portfolio | Digital Media Art

A creative digital media art portfolio website showcasing interactive installations, physical computing, and creative coding projects. Built with a focus on immersive user experience, featuring dynamic animations and a custom interactive dark mode.

## ✨ Features

- **Immersive Dark Mode:** Default dark theme featuring a dynamic, interactive aurora background (`data-theme="dark"`).
- **Smooth Animations:** Powered by GSAP for seamless page transitions, content reveals, and typographic effects.
- **Interactive Project Panels:** Custom UI/UX design with immersive full-screen project overlays, replacing traditional scrolling mechanics.
- **Bilingual Interface:** Structured to support English and Italian (EN/IT) for international academic applications.
- **One-Click Download:** Integrated high-resolution PDF portfolio download for quick offline access.

## 🛠 Tech Stack

- **Build Tool:** [Vite](https://vitejs.dev/) (Vanilla JS setup for optimal performance)
- **Animation Engine:** [GSAP](https://gsap.com/)
- **Styling:** Custom CSS3 (CSS Variables, Backdrop-filter, Flexbox/Grid)
- **Deployment:** [Vercel](https://vercel.com/) (Edge Network with custom CSP configuration)

```bash
